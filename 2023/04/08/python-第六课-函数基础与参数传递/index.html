特性 ❌ 错误写法box=[] ✅ 正确写法box=None
默认值是什么？ 一个具体的列表对象 [] 一个空值标记 None
何时创建列表？ 定义函数时（只创建1次，终身复用） 调用函数时（每次调用都创建）
多次调用后果 所有调用共享同一个列表，数据会累积 每次调用都是独立的新列表，互不干扰
